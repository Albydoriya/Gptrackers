# Supplier Logos Directory

This directory contains logo images used in Excel export templates.

## Required Files

### HPI Logo
- **Filename:** `hpi-logo.png`
- **Dimensions:** 150x50 pixels (or similar aspect ratio)
- **Format:** PNG with transparency
- **Max Size:** 50KB
- **Usage:** Used in HPI supplier Excel templates

### Company Logo
- **Filename:** `company-logo.png`
- **Dimensions:** 200x60 pixels
- **Format:** PNG with transparency
- **Max Size:** 50KB
- **Usage:** Used in header of all Excel templates

## Adding New Supplier Logos

1. Save logo as PNG with transparency
2. Name file: `{supplier-name}-logo.png` (lowercase, hyphenated)
3. Recommended size: 150x50 to 200x60 pixels
4. Update supplier record in database with logo path

## Notes

- Logos are embedded directly in Excel files
- Keep file sizes small for faster generation
- Use transparent backgrounds for professional appearance
